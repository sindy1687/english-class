🎬 MP4 作為 Image 屬性測試

← 返回卡片倉庫 🧪 一般 MP4 測試

📋 新功能說明

現在您可以直接在 image 屬性中使用 MP4 檔案路徑，卡片會直接顯示影片而不是圖片！

{ "word": "dancing_cat", "zh": "跳舞的貓", "image": "sound/dancing_cat.mp4", // 直接使用 MP4 檔案 "rarity": "稀有", "category": "可愛動物", "description": "一隻可愛的貓咪在跳舞" }

🎯 功能特點

✅ 直接顯示影片

MP4 檔案直接作為卡片的主要媒體顯示，無需額外的影片按鈕

🔄 自動播放循環

已解鎖的卡片會自動播放並循環影片

🎮 播放控制

點擊控制按鈕可以暫停/播放影片

🔒 未解鎖處理

未解鎖時顯示影片第一幀作為縮圖

📝 使用方法

方法一：image 屬性（推薦）

將 MP4 檔案路徑直接放在 image 屬性中

"image": "sound/your_video.mp4"

方法二：video 屬性

將 MP4 檔案路徑放在 video 屬性中

"video": "sound/your_video.mp4"
